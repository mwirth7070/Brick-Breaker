Remarks

This is Hack React. It is a Javascript version of the old Atari game Breakout. It should play in any browser that supports the HTML 5 CANVAS tag. This was inspired by a project from Jack Palevich, which can be found here.

Modifications from the original include the following:

  • Difficulty modes
  • Power ups
  • Enhanced graphics
  • Custom brick layout functionality
  • Start game input and animation
  • Color scheme and styling
  • Ball shape and behavior
  • Embedded score and ball count elements into canvas
  • User launch interface
  • Grid and tile sizes
  • Destruction sound and visual effects
  • End of game routine
  • Additional documentation
